# TopStories 🧾

-   🚀 Install Packages

```bash
#To install packages
yarn install

#To install pod installa
npx pod-install ios

```

-   📱 iOS

```bash
#To run the app 🏃‍♂️
yarn ios

#To run on iPhone 14 simulator 🏃‍♂️
yarn ios:14

#To clean ios 🚨
yarn clean:ios

```

-   ☎️ Android

```bash
#To run on android 🏃‍♂️
yarn android

#To build android release apk 🛠️
yarn android:release

#To clean android 🚨
yarn clean:android
```
